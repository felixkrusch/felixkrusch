Kunden empfehlen. 625 € verdienen.

25 % Provision — ausgezahlt am selben Tag, sobald der Kunde vollständig bezahlt hat

So funktioniert es (dauert nur 60 Sekunden)

  • Sagen Sie dem Kunden: 'Ich habe dich an Felix weiterempfohlen, schau dir seine Website und Referenzen auf felixkrusch.com an — schreib ihm direkt an fk@felixkrusch.com und erwähne meinen Namen.'
  • Schreiben Sie mir eine E-Mail an fk@felixkrusch.com und teilen Sie mir mit:

    Empfehlung: Kunde [Firmenname] wird sich bei Ihnen für eine neue Website melden. Das ist meine Empfehlung.

  • Der Kunde meldet sich bei mir → Projekt abgeschlossen schnell → Sie erhalten 625 € (25 % des Gesamtbetrags) innerhalb von 24 Stunden nach Zahlung des Kunden.

Regeln (ganz einfach)

  • Nur neue Kunden
  • Projekt ab 2.500 €
  • Passt zu meinem Workflow (kein WordPress)

Leiten Sie diese Seite einfach an Unternehmen weiter, die eine neue Website brauchen — oder teilen Sie dieses Empfehlungsprogramm mit allen, von denen Sie denken, dass sie damit gutes Geld verdienen könnten.
